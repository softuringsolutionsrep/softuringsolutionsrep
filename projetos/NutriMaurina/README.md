## Sobre o projeto

Site institucional de uma nutricionista com visual elegante, acolhedor e focado em conversão. O projeto foi pensado para apresentar autoridade, gerar confiança e facilitar o contato com a paciente em poucos cliques.

**Página oficial:** https://nutrimaurina.com.br

## Destaques

- Hero com identidade visual premium
- Seções sobre, serviços, abordagem, depoimentos, FAQ e CTA final
- Mini anamnese interativa em 5 etapas
- Envio de contato via WhatsApp
- Layout responsivo e acessível
- Paleta terrosa com detalhes em dourado

## Stack usada

- React
- Vite
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React
- React Hook Form
- Zod
- Embla Carousel
- Wouter
